### 目标

实现人脸认证的简单demo

### 技术栈

#### 服务端
- Java Spring Boot
- MySQL

#### 浏览器客户端
- Vue3 + Vite

#### 电脑客户端
- Electron


### 功能需求

#### 服务端功能：
1. 维护人脸数据（创建，更新，删除）
2. 校验人脸数据

#### 浏览器客户端功能：
1. 维护人脸数据的前端界面

#### 电脑客户端功能：
1. 采集人脸数据
2. 调用服务端进行人脸校验


### 任务分解

在face-auth-web目录下创建前端工程
1. 使用vite+vue3+naive-ui技术栈
2. 只需要有一个列表页面，展示注册的人脸数据
3. 列表页面有新增，编辑，删除功能
4. 新增和编辑功能需要有一个表单，表单包含用户ID，人脸图片

在face-auth-server目录下创建后端工程
1. 使用Spring Boot技术栈
2. 设计人脸数据表结构
3. 实现人脸数据的增删改查接口
4. 使用sqlite数据库
