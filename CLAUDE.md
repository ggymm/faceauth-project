### 目标

实现人脸认证的简单demo

### 技术栈

#### 服务端

- Java Spring Boot
- SQLite
- Mybatis-Plus

#### 浏览器

- Vue3 + Vite
- Naive-UI
- Axios
- Unocss

#### 客户端

- Electron

#### 模型服务

- Python Flask
- InsightFace
- ONNX Runtime

### 调试启动

1. 启动模型服务
2. 启动服务端
3. 启动浏览器端
4. 启动客户端

### 操作流程

1. 拍摄人脸照片
2. 通过客户端或者管理后台注册人脸
3. 使用客户端拍摄人脸进行认证

### 内部逻辑

1. 注册人脸时，将图片数据发送到模型服务，提取人脸特征向量，存储到数据库
2. 认证人脸时，同样将图片数据发送到模型服务，提取人脸特征向量
3. 将提取到的特征向量与数据库中的特征向量进行对比，计算相似度
4. 如果相似度超过设定阈值，则认证通过，否则认证失败
